# 微信第三方平台接入（component_appid/component_appsecret）

本文件仅说明微信开放平台第三方授权的接入流程，与部署 SOP 无关。

## 1. 官方文档

- 微信开放平台（第三方平台）文档索引  
  https://open.weixin.qq.com/cgi-bin/showdocument?action=dir_list&t=resource/res_list&verify=1&id=open1419318590
- 第三方平台接入指南（Standard Flow B）  
  https://open.weixin.qq.com/cgi-bin/showdocument?action=doc&id=open1419318590&t=resource/res_list

> 提示：上述链接在未登录或未完成企业认证的账号下可能显示空白或无内容。  
> 请在已登录开放平台账号的浏览器中打开，或从开放平台后台“文档中心/资源列表”进入对应条目。

## 2. 开放平台后台需要配置

- component_appid / component_appsecret
- component_token / component_aes_key
- 授权事件接收URL：`https://paiban.lx05.art/api/wechat/ticket`
- 授权回调地址（前端）：`https://paiban.lx05.art/settings/wechat/callback`

## 3. 服务器环境变量

后端（运行期）`/root/paiban/.env`：
```
WECHAT_COMPONENT_APP_ID=你的component_appid
WECHAT_COMPONENT_APP_SECRET=你的component_appsecret
# 兼容字段（可选）
WECHAT_OPEN_APP_ID=你的component_appid
WECHAT_OPEN_APP_SECRET=你的component_appsecret
```

前端（构建期）`/root/paiban/.env.production`：
```
VITE_APP_DOMAIN=https://paiban.lx05.art
VITE_WECHAT_OPEN_APP_ID=你的component_appid
```

> 修改 `.env.production` 后必须重建前端容器。

## 4. 接口与代码位置

### 后端接口（NestJS）
- 预授权码：`POST /api/wechat/pre-auth-code`  
  代码：`content-backend/src/wechat/wechat.controller.ts`
- 授权码换取令牌：`POST /api/wechat/exchange-auth`  
  代码：`content-backend/src/wechat/wechat.controller.ts`
- 已授权账号列表：`GET /api/wechat/authorized-accounts`  
  代码：`content-backend/src/wechat/wechat.controller.ts`
- 图片上传：`POST /api/wechat/upload`  
  代码：`content-backend/src/wechat/wechat.controller.ts`
- 草稿创建：`POST /api/wechat/draft`  
  代码：`content-backend/src/wechat/wechat.controller.ts`
- VerifyTicket 接收：`POST /api/wechat/ticket`  
  代码：`content-backend/src/wechat/wechat.controller.ts`

### 后端核心逻辑
- 组件 AccessToken 与授权信息：`content-backend/src/wechat/wechat.service.ts`
- 授权账号表：`content-backend/src/entities/wechat-authorizer.entity.ts`
- 平台配置表：`content-backend/src/entities/wechat-platform-config.entity.ts`

### 前端授权入口
- 授权管理组件：`src/components/WechatAuthManager.vue`
- 授权回调页面：`src/views/Settings/WechatCallback.vue`
- 授权服务：`src/services/ThirdPartyWechatAuth.ts`

## 5. 验证流程

1. 前端进入：`/settings/wechat` → “公众号授权”
2. 点击“添加新公众号授权”
3. 生成授权链接并扫码
4. 回调成功后，授权列表出现新公众号

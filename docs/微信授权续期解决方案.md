# 微信API授权续期解决方案

## 核心问题
微信OAuth2授权的refresh_token有效期仅为30天，无法延长。用户30天后必须重新手动授权。

## 135编辑器可能的解决方案

### 方案1: 公众号服务端授权 ⭐⭐⭐⭐⭐

这是最可能的解决方案，完全绕过了30天限制：

**原理：**
- 不使用网页授权（snsapi_base/snsapi_userinfo）
- 使用公众号服务端API授权
- 通过公众号应用凭证获取永久access_token

**实现步骤：**

1. **微信公众号设置**
```bash
# 在微信公众平台配置
1. 申请服务号（企业认证）
2. 开发者中心 -> 基本配置 -> 服务器配置
3. 配置服务器URL、Token、EncodingAESKey
4. 启用服务器配置
```

2. **服务端获取永久access_token**
```javascript
// 服务端代码 - Node.js示例
const axios = require('axios');

class WechatService {
  constructor() {
    this.appId = 'your_app_id';
    this.appSecret = 'your_app_secret';
    this.accessToken = null;
    this.tokenExpireTime = 0;
  }

  // 获取永久access_token（2小时有效，可无限刷新）
  async getAccessToken() {
    const now = Date.now();

    // 提前5分钟刷新
    if (this.accessToken && now < this.tokenExpireTime - 5 * 60 * 1000) {
      return this.accessToken;
    }

    const url = `https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${this.appId}&secret=${this.appSecret}`;

    const response = await axios.get(url);
    const { access_token, expires_in } = response.data;

    this.accessToken = access_token;
    this.tokenExpireTime = now + expires_in * 1000;

    return access_token;
  }

  // 使用access_token调用微信API
  async uploadImage(imageData) {
    const token = await this.getAccessToken();

    const formData = new FormData();
    formData.append('media', imageData);

    const response = await axios.post(
      `https://api.weixin.qq.com/cgi-bin/media/upload?access_token=${token}&type=image`,
      formData
    );

    return response.data;
  }
}
```

3. **前端无需用户授权**
```javascript
// 前端代码 - 完全不需要用户授权
class WechatImageUploader {
  async uploadImage(file) {
    // 直接调用后端API，无需任何微信授权
    const formData = new FormData();
    formData.append('image', file);

    const response = await fetch('/api/wechat/upload', {
      method: 'POST',
      body: formData
    });

    return response.json();
  }
}
```

**优势：**
- ✅ 无需用户授权
- ✅ 无30天限制
- ✅ 无需refresh_token
- ✅ 永久可用
- ✅ 符合微信规范

**限制：**
- 需要有认证的服务号
- 只能操作公众号本身的内容
- 无法获取用户个人信息

### 方案2: 多应用授权轮换 ⭐⭐⭐

**原理：**
- 创建多个微信应用
- 当一个应用的授权即将过期时，切换到另一个
- 用户只需要授权一次，后续自动切换

**实现步骤：**

1. **创建多个应用**
```javascript
const WECHAT_APPS = [
  { appId: 'app1_id', appSecret: 'app1_secret', name: '主应用' },
  { appId: 'app2_id', appSecret: 'app2_secret', name: '备用应用1' },
  { appId: 'app3_id', appSecret: 'app3_secret', name: '备用应用2' },
];
```

2. **授权切换逻辑**
```javascript
class WechatAuthManager {
  constructor() {
    this.currentAppIndex = 0;
    this.apps = WECHAT_APPS;
  }

  async getValidAuth() {
    // 检查当前应用是否有效
    if (await this.isAppValid(this.currentAppIndex)) {
      return this.apps[this.currentAppIndex];
    }

    // 寻找下一个有效的应用
    for (let i = 0; i < this.apps.length; i++) {
      if (await this.isAppValid(i)) {
        this.currentAppIndex = i;
        return this.apps[i];
      }
    }

    // 没有有效应用，需要重新授权
    throw new Error('所有应用都需要重新授权');
  }

  async isAppValid(index) {
    const app = this.apps[index];
    const authData = this.getStoredAuth(app.appId);

    if (!authData) return false;

    // 检查是否在30天内
    const now = Date.now();
    const authTime = authData.timestamp;
    const daysSinceAuth = (now - authTime) / (24 * 60 * 60 * 1000);

    return daysSinceAuth < 25; // 提前5天切换
  }
}
```

### 方案3: 智能提醒+一键续期 ⭐⭐

**原理：**
- 接受30天限制
- 在即将过期时智能提醒
- 提供一键续期按钮，简化用户操作

**实现步骤：**

1. **智能提醒系统**
```javascript
class WechatRenewalReminder {
  checkAndRemind(userAuth) {
    const now = Date.now();
    const authTime = userAuth.timestamp;
    const daysUntilExpiry = 30 - (now - authTime) / (24 * 60 * 60 * 1000);

    // 不同时间点的提醒策略
    if (daysUntilExpiry <= 3) {
      this.showCriticalReminder();
    } else if (daysUntilExpiry <= 7) {
      this.showWarningReminder();
    } else if (daysUntilExpiry <= 15) {
      this.showGentleReminder();
    }
  }

  showGentleReminder() {
    this.showNotification({
      title: '微信授权提醒',
      message: '您的微信授权将在15天后过期，请及时续期',
      type: 'info'
    });
  }

  showCriticalReminder() {
    this.showModal({
      title: '微信授权即将过期',
      message: '您的微信授权将在3天后过期，请立即续期以免影响使用',
      actions: [
        { text: '立即续期', primary: true, action: () => this.startRenewal() },
        { text: '稍后', action: () => this.scheduleReminder() }
      ]
    });
  }
}
```

**实施步骤：**

1. **申请微信公众号服务号**

1. 关注公众号（一次性操作）
2. 永久使用服务，无需任何授权续期

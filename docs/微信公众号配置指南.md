# 微信公众号配置指南

## 目标
实现无用户授权、无30天限制的微信API调用。

## 前提条件
1. **企业认证的服务号**（个人订阅号无法使用服务端API）
2. **已开通微信支付**（可选，用于商业用途）

## 配置步骤

### 第1步：注册微信服务号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击"立即注册" → 选择"服务号"
3. 选择"企业"类型（需要企业营业执照）
4. 完成企业认证（需要300元/年认证费）

### 第2步：获取开发者信息

1. 登录微信公众平台
2. 进入"开发" → "基本配置"
3. 记录以下信息：
   - **AppID**（应用ID）
   - **AppSecret**（应用密钥）

### 第3步：配置服务器（可选，如需接收微信消息）

1. 在"基本配置"中点击"修改配置"
2. 填写服务器信息：
   - URL: `https://your-domain.com/api/wechat/callback`
   - Token: 自定义验证token（如：your_token_123）
   - EncodingAESKey: 随机生成
   - 消息加解密方式：选择"明文模式"

### 第4步：配置白名单

1. 在"基本配置"中找到"IP白名单"
2. 添加服务器IP地址：
   - 开发环境：添加你的IP
   - 生产环境：添加服务器IP

## 环境变量配置

### 后端环境变量
```bash
# .env 文件
WECHAT_APP_ID=your_app_id_here
WECHAT_APP_SECRET=your_app_secret_here
```

### 前端无需配置
前端完全不需要配置微信相关的环境变量。

## API权限说明

使用服务端API，无需用户授权即可调用：

✅ **可调用的API：**
- 获取access_token
- 上传素材（图片、语音、视频等）
- 创建/发布图文消息
- 获取关注者列表
- 发送客服消息
- 自定义菜单管理

❌ **无法调用的API：**
- 获取用户个人信息
- 网页授权相关API
- 微信支付相关API（需要额外配置）

```javascript
// 前端代码 - 用户操作
```

## 优势对比

### 传统网页授权 ❌
- 用户需要手动授权
- refresh_token只有30天有效期
- 每次授权需要用户确认
- 用户体验差

### 公众号服务端授权 ✅
- **零用户授权**
- **无30天限制**
- **永久可用**
- **用户体验最佳**
- **符合微信规范**

## 常见问题

### Q: 为什么需要企业认证的服务号？
A: 只有企业认证的服务号才有完整的服务端API权限。个人订阅号无法使用。

### Q: 需要用户关注公众号吗？
A: 不需要。服务端API与用户是否关注公众号无关。

### Q: 可以获取用户个人信息吗？
A: 不可以。如需用户信息，仍需使用网页授权。

### Q: access_token会过期吗？
A: 会，每2小时过期，但可以无限刷新，对用户透明。

## 验证配置

配置完成后，可以通过以下方式验证：

```bash
curl "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=YOUR_APPID&secret=YOUR_APPSECRET"
```

成功返回示例：
```json
{
  "access_token": "ACCESS_TOKEN",
  "expires_in": 7200
}
```

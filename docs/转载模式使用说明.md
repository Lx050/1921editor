# 转载模式 - 微信文章抓取功能

## 📋 功能概述

转载模式允许用户通过输入微信公众号文章链接，自动抓取并转换文章内容为本地格式。抓取的内容会自动添加三下乡模式的头部和尾部常量。

## 🎯 使用方法

### 1. 选择转载模式

在文章配置页面（`/article-config/:id`）选择「转载模式」。

### 2. 进入 Step 1 输入页面

系统会自动跳转到 Step 1 文本输入页面，此时会显示「转载模式 - 微信文章链接」输入区域。

### 3. 粘贴微信文章链接

1. 复制微信公众号文章链接（格式：`https://mp.weixin.qq.com/s/...`）
2. 粘贴到输入框中
3. 点击「📥 抓取文章内容」按钮或按 Enter 键

### 4. 自动转换

系统会自动：
- 抓取文章标题、作者和正文内容
- 将 HTML 转换为自定义文本格式
- 自动提取图片位置（使用 `&` 标记）
- 将作为源公众号信息填充到尾部「来源：{{SOURCE_ACCOUNT}}」字段

### 5. 继续编辑

转换完成后，内容会自动填充到文本输入框，您可以：
- 继续编辑文本内容
- 点击「下一步：编辑内容」进入 Step 2
- 在 Step 3 预览时，头部和尾部会自动使用转载模式的常量

## 🔧 技术实现

### 后端 API

#### Endpoint: `POST /api/wechat/fetch-article`

**请求体：**
```json
{
  "url": "https://mp.weixin.qq.com/s/..."
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "html": "<div>文章内容...</div>",
    "title": "文章标题",
    "author": "公众号名称"
  }
}
```

### 前端实现

#### 文件修改

1. **后端服务** (`content-backend/src/wechat/wechat.service.ts`)
   - 添加 `fetchArticleFromUrl()` 方法
   - 使用 axios 抓取微信文章
   - 正则表达式提取标题、作者和正文

2. **后端控制器** (`content-backend/src/wechat/wechat.controller.ts`)
   - 添加 `/fetch-article` endpoint

3. **前端 API** (`src/utils/api.ts`)
   - 添加 `wechatApi.fetchArticle()` 方法

4. **Step1 输入页面** (`src/views/Step1TextInput.vue`)
   - 添加微信链接输入区域
   - 添加 `fetchWechatArticle()` 方法
   - 集成 `convertHtmlToCustomFormat()` 转换器

5. **配置Store** (`src/stores/configStore.ts`)
   - 已有转载模式头部和尾部常量（`REPRINT_HEADER`, `REPRINT_FOOTER`）

## 📝 模式配置

### 头部常量 (REPRINT_HEADER)

使用与日常模式相同的顶部图片和基础结构。

### 尾部常量 (REPRINT_FOOTER)

尾部信息包含：
- **来源：{{SOURCE_ACCOUNT}}** - 自动填充原公众号名称
- **编辑：{{EDITOR_INPUT}}** - 手动输入编辑者姓名
- **审核：王雪 宋欣翼**
- **责编：朱梦鹤**
- 底部二维码图片

## 🚀 工作流程

```
用户输入微信链接
    ↓
后端抓取文章内容
    ↓
提取标题、作者、正文
    ↓
HTML → 自定义文本格式转换
    ↓
自动填充到文本输入框
    ↓
用户继续编辑 (Step 2, Step 3)
    ↓
应用转载模式头尾部
    ↓
导出/发布到微信
```

## ⚠️ 注意事项

1. **链接有效性**：只支持 `mp.weixin.qq.com` 域名的文章链接
2. **网络要求**：需要服务器能够正常访问微信服务器
3. **内容准确性**：抓取的内容可能需要手动调整格式和图片位置
4. **图片处理**：抓取时只标记图片位置，实际图片需要在 Step 2 重新上传
5. **版权声明**：使用转载模式时，请确保已获得原作者授权

## 🐛 常见问题

### 1. 抓取失败

**原因**：
- 链接不正确
- 网络问题
- 微信反爬虫策略

**解决方案**：
- 检查链接格式
- 稍后重试
- 联系管理员检查服务器网络

### 2. 内容格式混乱

**原因**：
- HTML 结构复杂
- 特殊排版元素

**解决方案**：
- 在 Step 2 手动调整内容块
- 重新标记图片位置

### 3. 作者信息缺失

**原因**：
- 文章页面结构变化
- 正则表达式未匹配

**解决方案**：
- 手动在 Step 3 中填写来源公众号

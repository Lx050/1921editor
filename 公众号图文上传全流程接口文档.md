# 公众号图文上传全流程接口文档

## 接口概述

本文档包含 “实时获取 Token→上传图片→创建图文草稿→获取预览二维码” 全流程 4 个核心接口，适用于已认证公众号（订阅号 / 服务号），无需依赖 Python 环境，可通过 Postman、curl 等工具直接调用。

## 前置条件



1. 公众号 APPID、APPSECRET（获取路径：公众号后台→「设置与开发」→「基本配置」）

2. 调用方 IP 已添加到公众号 IP 白名单（配置路径：公众号后台→「开发」→「基本配置」→「IP 白名单」）

3. 待上传图片：格式 png/jpg，大小≤2MB

4. 正式号需准备关注者 openid（用于预览接口接收者）

## 核心接口清单



| 接口名称       | 接口 URL                                                    | 请求方式 | 核心作用                           |
| ---------- | --------------------------------------------------------- | ---- | ------------------------------ |
| 实时获取 Token | `https://api.weixin.qq.com/cgi-bin/stable_token`          | POST | 获取有效 access\_token（有效期 7200 秒） |
| 上传图片（永久素材） | `https://api.weixin.qq.com/cgi-bin/material/add_material` | POST | 上传本地图片到微信服务器，返回 media\_id      |
| 创建图文草稿     | `https://api.weixin.qq.com/cgi-bin/draft/add`             | POST | 合成文字 + 图片，创建公众号图文草稿            |
| 获取预览二维码    | `https://api.weixin.qq.com/cgi-bin/draft/preview`         | POST | 生成图文预览链接，扫码即可预览                |

## 接口调用流程（按顺序执行）



1. 调用「实时获取 Token 接口」，获取`access_token`；

2. 调用「上传图片接口」，传入`access_token`和本地图片，获取`media_id`和`image_url`；

3. 调用「创建图文草稿接口」，传入`access_token`、`media_id`和文章内容，获取`draft_id`；

4. 调用「获取预览二维码接口」，传入`access_token`和`draft_id`，获取`url`；

5. 将`url`生成二维码（可通过在线二维码生成工具：[https://cli.im/](https://cli.im/) ），手机微信扫码预览。

## 关键注意事项



1. Token 有效期 7200 秒，过期后需重新调用接口获取，无需频繁刷新；

2. 图片 media\_id 为永久素材，可重复使用（删除路径：公众号后台→「素材管理」→「图片」）；

3. 预览链接有效期 24 小时，过期需重新调用「获取预览二维码接口」生成；

4. 正文 HTML 支持的标签：`<p>``<strong>``<em>``<br>``<img>``<a>`等，不支持 JavaScript；

5. 正式号必须使用真实关注者的 openid（获取接口：`https://api.weixin.qq.com/cgi-bin/user/get?access_token=xxx`），否则预览失败。

## 常见错误码速查



| 错误码   | 错误说明               | 解决方案                             |
| ----- | ------------------ | -------------------------------- |
| 40001 | invalid credential | Token 过期或无效，重新获取 Token           |
| 40164 | invalid ip         | IP 未添加到白名单，在公众号后台配置 IP 白名单       |
| 40007 | invalid media\_id  | media\_id 无效，重新上传图片获取新 media\_id |
| 41005 | media data missing | 图片文件未上传或格式错误，检查图片路径和格式           |
| 40037 | invalid draft\_id  | draft\_id 无效，重新创建图文草稿            |
| 40013 | invalid appid      | APPID 错误，核对公众号后台的 APPID          |
